🌍 **Analisi Qualità dell'Aria Italia (2001-2022)**
-----------------------------------------------------------------------------
Un progetto Python per l'analisi statistica e la visualizzazione geografica dei dati sull'inquinamento atmosferico in Italia, basato su dataset storici di O3, NO2, PM2.5 e PM10.

✨ **Funzionalità**
-------------------------------------------------------------------------------------------------------------------
- Analisi Statistica: Elaborazione automatica di medie, minimi, massimi e deviazione standard per regione e anno.
- Mappa Interattiva: Generazione di una mappa geografica (Folium) con marker cliccabili per ogni regione.
- Visualizzazione Dati: Generazione automatica di oltre 1.500 grafici a barre in formato PNG.
- Report HTML: Creazione di una dashboard interattiva e di una pagina di dettaglio per consultare tutti i grafici prodotti.

🛠️ **Tecnologie Utilizzate**
-----------------------------------------------------------------------
*   Python 3.12
*   Pandas: Per la manipolazione e pulizia dei dati CSV.
*   Matplotlib: Per la generazione dei grafici statistici.
*   Folium: Per la creazione della mappa interattiva in HTML/JavaScript.

📦 **Struttura del Progetto**
--------------------------------------------------------------------------
*   `Tesi.ipynb`: Lo script principale che elabora i dati e genera l'output.
*   `Grafici Statistiche`: Cartella contenente tutti i grafici PNG generati.
*   `Mappa Qualità Aria Con Grafici.html`: La mappa interattiva principale.
*   `Grafici In Dettaglio.html`: Galleria completa dei grafici generati.
